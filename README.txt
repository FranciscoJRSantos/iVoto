Para correr os ficheiros .jar basta utilizar o comando java -jar filename.jar. Não é necessário correr com nenhum argumento a não ser o TCPClient, se se quiser especificar o IP e porto (assume localhost e port 12345 caso contrário). Todos os outros programas têm endereços e porto alteráveis nos ficheiros .proprieties, assim como outras configurações explicitadas na página 10 do relatório
É necessário ter o MySQL 5.7 instalado para o funcionamento correcto do servidor RMI. É fornecido um script SQL (ivotobd.sql) que pode ser usado para popular a base de dados com algumas informações.
